# 🛒 Spring MVC E-Commerce Application

ProjectDemo video Link : https://techouts2-my.sharepoint.com/:v:/g/personal/manoj_m_techouts_co_in/IQC7k2l9dnFKRYNVmLwMes7bAVantBWWwKppVN74zNPQVTM?e=OmbEs7

A full-featured E-Commerce web application built using Spring MVC.  
This project demonstrates product management, user authentication, cart functionality, and order processing using Java-based backend technologies.

---

## 🚀 Features

- 👤 User Registration & Login
- 🛍️ Product Listing & Details Page
- 🛒 Add to Cart / Remove from Cart
- 📦 Order Placement
- 🔐 Spring Security Authentication
- 🗄️ Database Integration (MySQL)
- 🧾 Admin Panel for Product Management

---

## 🛠️ Technologies Used

- Java 8+
- Spring MVC
- Spring Security
- Hibernate / JPA
- JSP & JSTL
- MySQL
- Maven
- Apache Tomcat
