A full-featured E-Commerce web application built using Spring MVC. This project demonstrates product management, user authentication, cart functionality, and order processing using Java-based backend technologies.

🚀 Features

  • 👤 User Registration & Login
  • 🛍️ Product Listing & Details Page
  • 🛒 Add to Cart / Remove from Cart
  • 📦 Order Placement
  • 🔐 Spring Security Authentication
  • 🗄️ Database Integration (PostgreSQL)
  • 🧾 Admin Panel for Product Management

🛠️ Technologies Used

  • Java 8+
  • Spring MVC
  • Spring Security
  • Hibernate / JPA
  • JSP & JSTL
  • PostgreSQL
  • Maven
  • Apache Tomcat
